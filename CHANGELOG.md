# Changelog

All notable changes to CodexStats are documented here.

## [0.1.0] - 2026-05-08

### Added

- Initial distributable Electron build for Codex subscription usage monitoring.
- Local Codex session log scanner for `~/.codex/sessions/**/*.jsonl`.
- Full-screen dashboard with 5-hour and 7-day usage gauges.
- Recent sessions table, session detail modal, historical charts, and export tools.
- Optional OpenAI organization costs sync through the official `/v1/organization/costs` endpoint.
- GitHub release publishing and packaged-app auto-update checks.
- Windows NSIS installer icon and app icon assets.
